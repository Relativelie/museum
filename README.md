# Museum site 

Used: 
- typescript
- react
- scss
- unit tests

Online demo: https://relativelie.github.io/museum/

## How to build
1. Clone repository
``git clone git@github.com:Relativelie/museum.git /path/to/museum``
``cd  /path/to/museum ``
2. Install dependencies
``npm install ``
3. Build project
``npm run deploy``

## How to start
1. Run project 
``npm start ``
