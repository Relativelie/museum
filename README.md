# Museum site 

Used: 
- typescript
- react
- scss

## How to build
1. Clone repository
``git clone git@github.com:Relativelie/museum.git /path/to/map``
``cd  /path/to/map ``
2. Install dependencies
``npm install ``
3. Build project
``npm run deploy``

## How to start
1. Run project 
``npm start ``
